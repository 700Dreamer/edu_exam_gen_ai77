# EduQuest AI Engine

This repository contains the backend and frontend for the EduQuest AI educational engine.

## 📁 Project Structure
- `/` (Root): Backend logic (FastAPI), deployment scripts, and configuration.
- `/frontend`: Next.js web application.
- `/core`: Core AI and data processing modules.

## 🛠 DevOps & Deployment
Before making changes to Docker files, shell scripts, or deployment configurations, please read the **[DevOps Guide](./DEVOPS_GUIDE.md)**.

## 🚀 Quick Start
To run the full stack locally using Docker:
```bash
docker compose up --build
```

For detailed instructions, see the guide mentioned above.
